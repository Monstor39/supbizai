## 📑 СТРУКТУРА САЙТА (8 секций)

### 1. HERO (Главный экран)
- **Заголовок:** "Ваш бизнес работает пока вы спите"
- **Подзаголовок:** "Автоматизирую процессы которые съедают ваше время и деньги"
- **Элементы:**
  - Badge с иконкой CPU
  - 2 CTA кнопки: "Получить бесплатный аудит" + "Смотреть кейсы"
  - Анимированный фон: grid-сетка + частицы + градиентные пятна

### 2. БОЛИ КЛИЕНТА (Pain Points)
- **Заголовок:** "Узнаёте себя?"
- **6 flip-карточек:** При клике переворачиваются, показывают решение
- **Боли:**
  1. Менеджеры тратят 4 часа на одинаковые ответы
  2. Данные теряются при переносе между таблицами
  3. Заявки теряются из-за задержек
  4. Отчёты готовятся 2 дня
  5. Долгий онбординг сотрудников
  6. Ошибки стоят денег

### 3. КАК РАБОТАЕТ (Timeline)
- **3 шага:**
  1. АУДИТ → Разбираем процессы
  2. РЕШЕНИЕ → Внедряем автоматизацию
  3. РЕЗУЛЬТАТ → Система работает 24/7
- **Анимация:** Прогресс-линия рисуется при скролле

### 4. ROI КАЛЬКУЛЯТОР (Интерактив)
- **3 ползунка:**
  - Количество сотрудников (1-20)
  - Средняя зарплата (30-150 тыс ₽)
  - Часов в день на рутину (1-8)
- **Формула:**
  - Месячные потери = сотрудники × зарплата × (часы / 8)
  - Годовые потери = месячные × 12
  - Экономия = годовые × 0.85 (85% автоматизация)
- **Кнопка:** "Хочу сэкономить эти деньги"

### 5. КЕЙСЫ (Projects)
- **Кейс 1: AI-менеджер для китайской языковой школы**
  - Проблема: Менеджер не успевал обрабатывать заявки
  - Решение: AI-бот в Instagram/VK
  - Результаты: 24/7 работа, 0 потерь, -1 менеджер

- **Кейс 2: Система КРУ для предприятия**
  - Статус: В разработке
  - Плейсхолдер: "Скоро результаты"

### 6. СФЕРЫ ПРИМЕНЕНИЯ (Industries)
- **9 карточек 3×3:**
  - Розница, HoReCa, Онлайн-школы
  - Медицина, Логистика, Недвижимость
  - Производство, Маркетинговые агентства, E-commerce
- **Интерактив:** Tooltip при наведении с примером автоматизации

### 7. CTA (Призыв к действию)
- **Фон:** Градиент purple→blue
- **Заголовок:** "Первый шаг — бесплатно"
- **Форма:** Имя + Телефон/Telegram
- **Соцсети:** Telegram, WhatsApp, Email, LinkedIn

### 8. FOOTER

- Ссылки на соцсети

## 🔑 MCP СЕРВЕРЫ (Конфигурация)

**Файл:** `.vscode/mcp.json`

### Установленные серверы:
1. **filesystem** — Работа с файлами проекта
2. **fetch** — Проверка CDN и внешних ресурсов
3. **github** — Коммиты, деплой (требует GitHub токен)
4. **puppeteer** — Скриншоты, тестирование
5. **memory** — Память между сессиями
6. **openrouter** — AI через Qwen/Claude (требует API ключ)

### Команды установки:
```bash
npm install -g @modelcontextprotocol/server-filesystem
npm install -g @modelcontextprotocol/server-fetch
npm install -g @modelcontextprotocol/server-github
npm install -g @modelcontextprotocol/server-puppeteer
npm install -g @modelcontextprotocol/server-memory
```

### Скрипт установки:
```bash
.\install-mcp.bat
```

## 🆘 TROUBLESHOOTING

### MCP не работает:
```bash
# Проверить установку Node.js
node --version
npm --version

# Переустановить серверы
npm uninstall -g @modelcontextprotocol/server-filesystem
npm install -g @modelcontextprotocol/server-filesystem

# Перезапустить VS Code
```

### Сайт не открывается:
- Проверить что файл не заблокирован Windows
- Попробовать другой браузер
- Проверить консоль браузера (F12)
