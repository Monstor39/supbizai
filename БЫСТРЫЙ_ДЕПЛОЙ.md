# 🚀 БЫСТРЫЙ ДЕПЛОЙ — GitHub Pages

## ⚡ САМЫЙ ПРОСТОЙ СПОСОБ (без Git в консоли)

### Вариант 1: GitHub Desktop (рекомендую!)

**1. Скачайте GitHub Desktop:**
- Откройте https://desktop.github.com/
- Нажмите **Download for Windows**
- Установите

**2. Войдите в аккаунт GitHub:**
- Откройте GitHub Desktop
- Sign in через GitHub

**3. Добавьте проект:**
- File → **Add Local Repository**
- Выберите папку: `c:\Users\111\Desktop\autohub`
- Нажмите **Add repository**

**4. Опубликуйте на GitHub:**
- Нажмите **Publish repository**
- Имя: `supbizai`
- **✓ Public** (обязательно!)
- Нажмите **Publish**

**5. Включите GitHub Pages:**
- Откройте https://github.com/YOUR_USERNAME/supbizai/settings/pages
- **Source:** Deploy from a branch
- **Branch:** main → **root**
- Нажмите **Save**

**Готово!** Сайт доступен по:
```
https://YOUR_USERNAME.github.io/supbizai/
```

---

### Вариант 2: Через сайт GitHub (вообще без установки)

**1. Создайте репозиторий:**
- Откройте https://github.com/new
- Имя: `supbizai`
- **✓ Public**
- **НЕ** инициализируйте README
- Нажмите **Create repository**

**2. Загрузите файлы:**
- На странице репозитория нажмите **uploading an existing file**
- Перетащите ВСЕ файлы из папки `autohub` в браузер
- Нажмите **Commit changes**

**3. Включите GitHub Pages:**
- Settings → **Pages** (слева)
- **Source:** Deploy from a branch
- **Branch:** main → root
- **Save**

**Готово!** Через 1-2 минуты сайт доступен по:
```
https://YOUR_USERNAME.github.io/supbizai/
```

---

## 🌐 Настройка домена supbizai.ru

**После того как сайт заработает на GitHub Pages:**

### 1. В GitHub Pages Settings:
- Settings → Pages → **Custom domain**
- Введите: `supbizai.ru`
- Нажмите **Save**

### 2. В панели Reg.ru:
Откройте https://www.reg.ru/domain/ns/ и измените DNS:

**Удалите старые записи!**

**Добавьте 4 A-записи:**
| Тип | Имя | Значение |
|-----|-----|----------|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |

**Добавьте 1 CNAME запись:**
| Тип | Имя | Значение |
|-----|-----|----------|
| CNAME | www | YOUR_USERNAME.github.io |

### 3. Подождите 15-60 минут

DNS обновится и сайт откроется по https://supbizai.ru

3. Если всё ок — настраивайте домен

## 🆘 Если что-то не работает

**Git не установлен:**
- Используйте Вариант 2 (загрузка через сайт GitHub)

**404 ошибка на GitHub Pages:**
- Подождите 2-3 минуты
- Проверьте что в настройках Pages выбрана ветка **main**

**Домен не работает:**
- Проверьте DNS на https://dnschecker.org/
- Убедитесь что удалили старые DNS записи на Reg.ru

---

**ВРЕМЯ:** 5-10 минут на деплой + 15-60 минут на DNS
