# 🚀 Деплой на Netlify Drop (без Git!)

## Шаг 1: Откройте Netlify Drop

1. Откройте https://app.netlify.com/drop
2. Войдите через GitHub (или создайте аккаунт)

## Шаг 2: Перетащите папку

1. Откройте проводник с папкой `autohub`
2. Перетащите **всю папку** прямо в браузер на Netlify Drop
3. Подождите 10-20 секунд

## Шаг 3: Сайт опубликован!

Вы получите URL вида: `https://random-name.netlify.app`

## Шаг 4: Настройте домен

1. Нажмите **Domain settings**
2. **Add custom domain**
3. Введите: `supbizai.ru`
4. Нажмите **Add domain**

## Шаг 5: Обновите DNS на Reg.ru

В панели Reg.ru (https://www.reg.ru/support/) удалите старые записи и добавьте:

| Тип | Имя | Значение |
|-----|-----|----------|
| A | @ | 75.2.60.5 |
| CNAME | www | supbizai.netlify.app |

**Важно:** Удалите все старые A-записи перед добавлением новых!

## Шаг 6: Подождите 5-30 минут

DNS обновится и сайт будет доступен по https://supbizai.ru
